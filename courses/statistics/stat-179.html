STAT 179 - Introduction to Survival Analysis

Course Description

Topics to be discussed in this course include: basic quantities for survival data, nonparametric inferences for basic survival quantities, semiparametric proportional hazards model, inferences for parametric regression models.

Course Learning Outcomes

After completion of the course, the student should be able to:

  1. Demonstrate understanding of the basic concepts of survival data;
  2. Utilize the different statistical inferences useful only for time - to - event data;
  3. Apply statistical techniques appropriate only for survival data using statistical software ;
  4. Perform survival analysis on time – to – event data that occur in different fields
