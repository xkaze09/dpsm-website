STATISTICS

Course Credits: 3 Units

Prerequisites: Math 153, Stat 104

STAT 138 - Use of Statistical Software Packages

Course Description

This course covers the use of database software, spreadsheet and statistical software packages for data management.

Course Learning Outcomes

After completion of the course, the student should be able to:

  1. Demonstrate awareness of the different statistical software packages available;
  2. Manage to operate some statistical software packages such as SPSS, SAS, and R;
  3. Create, import, manipulate/edit, manage data using different statistical software packages; and
  4. Perform basic statistical analysis on these software packages
